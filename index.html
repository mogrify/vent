Vent

Vent is a browser-based Ajax chat application for ASP.NET.

Its goal is to be a simple, easy-to-deploy, easy-to-use chat room for people within a corporate Intranet.

Vent requires no database, and it uses Windows Integrated Authentication (NTLM), so all domain users can automatically use it with no additional configuration.

Features

  • Users identified by their Windows username
  • No extra passwords required
  • Dynamic list of joined users
  • Public and private messages
  • Sounds played for events
  • Extensible: built on web services

Requirements

  • IIS server with the .NET Framework 3.5
  • IE or Firefox for all users
  • Windows domain containing server, clients, and users

Setup

  1. Copy to an IIS virtual directory.
  2. Have users visit http://servername/directory/ in their browsers.

Vent uses jQuery and SoundManager.
